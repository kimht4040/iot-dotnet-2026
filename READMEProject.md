# 지역구별 맛집 탐색 애플리케이션


## 기능
1. 부산광역시 맛집 api 활용하여 맛집표시
2. 지역구별로 선택하여 리스트로 보여줌
3. 맛있어보이는 맛집 클릭시 카카오맵으로 연결
